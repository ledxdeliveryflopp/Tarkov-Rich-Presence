# Текущие цели(по приоритету)

1) Полный переход на deque search(выборка N строк с конца файла)
2) Фикс багов(переход с локи на локу в пве, лив из пати(под вопросом), regular -> pvp)
3) обработка запуска браузера
4) Определение когда игра включена и нет + обновление папки актуального лога
5) Апдейтер 
6) запаковка браузера
7) мб переход на работу в трею через PySide
8) Автозапуск(почти сделан)
9) Теоритический переход в PyPresence на самописную либу(по приколу :3)
