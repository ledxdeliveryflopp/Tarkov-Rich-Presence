# Discord Rich Presence for Escape From Tarkov


**Интеграция Discord rich presence для игры Escape From Tarkov.**

### Особенности
1) Обновления через приложение + установщик
2) Фоновая работа в трее
3) Автозапуск
4) Определение имени профиля + уровня + престижа + фракции
5) Определение режима игры
6) Подсчет игроков в пати

### Как использовать
1) Создайте в папке игры любую папку
2) Скачайте [установщик](https://github.com/ledxdeliveryflopp/Tarkov-Rich-Presence-Installer/releases)
3) Перенесите установщик в созданную папку
4) Запустите от имени администратора(если нужен автозапуск)
5) Дождитесь установки
6) Сконфигурируйте приложение(если надо)
7) Имеет изображения всех локаций игры

**settings.yml**
```yaml
settings:
  core:
    debug: False # лучше не тройгате
    deque_max_depth: 1 # не реализовано
    deque_search: false # не реализовано
    log_folder_path: ../Logs # путь к логам игры(по умолчанию такой путь)
    profiler: false # включить профилирование приложения(статиска по скорости работы и тд)
  language: ru # Язык который будет в статусе
  presence:
    refresh_time: 8 # Время между обновлениями статуса
    show_zero_prestige: false # Показывать ли в статусе нулевой престиж
```

**logger.yml**
```yaml
logger:
  level: TRACE # Уровень логирования
  file: eft-discord-rich-presence-{time}.log # Название лог файла
  diagnostic: True # Расширенная трассировка исключений
```
7) Запустите EFTDRP.exe


### Как узнать что вышло обновление
Приложение при запуске проверяет актуальность текущей версии и если есть новее, то в трее будет написано

![img.png](/git_static/new_update.png)

1) Можно нажать на эту кнопку и приложение запустит установщик которое обновит приложение

**Если нет новой версии**

![img.png](/git_static/actual_version.png)

2) Можно закрыть приложение и запустить установщик, он предложит обновить приложение
